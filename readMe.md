#目录结构
```

├── dome.html 案例
├── readMe.md 
├── xx.js 头文件

    ├─ conf.js  核心配置文件
    ├─ core
    ├   ├──  core.js  核心
    ├   └──  jq.js jq
    ├─module
    │   ├── msgbox.js
    │   ├── msgbox_conf.js
    │   ├── pageHelp.js       分页模块
    │   ├── pageHelp_conf.js  分页配置文件
    │   ├── readMe.md         编写模块须知
    │   ├── renderTools.js    渲染模块@后期扔了重新写个新的
    │   ├── renderTools_conf.js  渲染模块配置文件
    │   ├── tools.js        通用工具类
    │   └── tools_conf.js  工具类配置文件
    └─static_html_resources 静态html资源
        ├── pageHelpCss       具体模块
            └── css.css
        └── msgboxCss
